> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Tech Stack:** TypeScript, Vitest, MCP SDK in-memory client, pnpm workspace, repo-local `rtk` wrapper.


- draft tools keep `experimental draft tool` and mention draft/review boundaries plus no DB connection, no SQL execution, or no filesystem writes as applicable.
