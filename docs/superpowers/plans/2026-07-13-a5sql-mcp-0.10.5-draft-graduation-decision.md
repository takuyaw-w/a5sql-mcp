# a5sql-mcp 0.10.5 Dry Run / Draft Graduation Decision Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 0.10.x の release dry run を完了し、5つの生成補助 tool を 1.0.0 前の卒業対象と判断しつつ、0.10.5では現在の experimental runtime contract を維持する。

**Architecture:** 0.10.5 は runtime feature を追加せず、public guidance、ROADMAP、version metadata、regression guard だけを更新する。graduation に必要な runtime gap は decision matrix に固定し、0.11.0から0.13.0へ送る。

**Tech Stack:** TypeScript、Vitest、MCP TypeScript SDK、pnpm workspace、oxfmt、oxlint

## Global Constraints

- 実装基点は commit `2c031e3` の0.10.4成果を含む `feat/0.10.5-draft-graduation-decision` worktree とする。
- 5 tool すべてを 1.0.0 前の卒業対象とするが、0.10.5では `experimental draft tool` marker を外さない。
- tool 名、tool 数、input schema、tool profile、handler / generator output、draft helper の MCP `outputSchema` 公開状態を変更しない。
- `outputKind: "draft"`、review 前提、DB 非接続、SQL 非実行、ファイルシステム非書き込み、秘密情報マスク、untrusted content の境界を維持する。
- Mermaid と全 response variant の draft contract gap は0.10.5で修正せず、0.11.0または0.12.0へ残す。
- version metadata と schema version だけを `0.10.5` に lockstep 更新する。
- merge と push は行わない。
